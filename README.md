Selenium Webdriver Automation - SauceDemo ( Tugas perbaikan)

Project ini adalah framework automation testing untuk website [SauceDemo](https://saucedemo.com) menggunakan **Node.js**, **Selenium Webdriver**, dan **Mocha/Chai**. Framework ini menerapkan design pattern **Page Object Model (POM)** untuk memastikan kode mudah dipelihara dan rapi.

Fitur Utama
- Page Object Model (POM): Memisahkan antara logika tes, aksi, dan elemen halaman.
- Automated Login Scenarios**: 
  - Login berhasil dengan user valid.
  - Validasi pesan error untuk username salah.
  - Validasi pesan error untuk password salah.
  - Validasi pesan error untuk user yang terkunci (*locked_out_user*).
